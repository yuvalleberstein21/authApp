![Auth App Image](./passport/assets/swagger-AuthApp.png)

##  转转

- 专砖 转专转 注  + 住住
- 砖专转 session 注 `express-session`
- 转专转 注 Google OAuth 2.0
- 转注 API 注 Swagger UI
- 转 -TypeScript 注 专拽专 专转

- ##  转拽

- git clone https://github.com/yuvalleberstein21/authApp.git
- cd passport
- npm install
 ##  爪专 拽抓 .env
- MONGO_URI=your_mongodb_uri
- SESSION_SECRET=your_secret
- GOOGLE_CLIENT_ID=xxx
- GOOGLE_CLIENT_SECRET=xxx
- GOOGLE_CALLBACK_URL=http://localhost:8005/auth/google/callback

- ##  专爪转 砖专转
- npm start

- ##  转注 API
- Swagger: http://localhost:8005/api-docs
- Google Login: 专 驻驻 , 志 http://localhost:8005/login.html
